# MindAR — țintă Eldrem

## Ce lipsește (tu, o singură dată)

Pagina `/demos/mind/` are nevoie de:

**`eldrem-logomark.mind`**

## Cum îl generezi

1. Deschide: https://hiukim.github.io/mind-ar-js-doc/tools/compile
2. Upload **`eldrem-logomark.jpg`** (sau copia din `assets/images/`)
3. Verifică că punctele de feature acoperă logo-ul
4. Download → redenumește **`eldrem-logomark.mind`**
5. Pune fișierul **în acest folder** (`assets/mind/`)
6. Push pe GitHub

Fără `.mind`, demo-ul MindAR nu pornește tracking-ul.

## Fișiere

| Fișier | Rol |
|--------|-----|
| `eldrem-logomark.mind` | descriptor MindAR (obligatoriu) |
| `eldrem-logomark.jpg` | referință / print (opțional în repo) |
